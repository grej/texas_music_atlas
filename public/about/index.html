About Texas Music Atlas

Texas Music Atlas is a volunteer-led, open source project created by a music lover who wants to make it easier to discover songwriter festivals and venues across the state. All data lives in simple JSON files so contributors can keep the calendar fresh without touching a server.

Want to help improve the listings, visuals, or tooling? Fork the GitHub repository, open a pull request, or drop a note in the project issues. Every bit of shared knowledge keeps the scene thriving.

Get Your Event or Venue Listed

Know of a songwriter festival or venue that should be included? We'd love to hear about it! Email us at venues@texasmusicatlas.com with details about the event or venue, including dates, location, website, and a brief description.

Contributing

This project is open source and hosted on GitHub. Whether you're a developer, designer, or music enthusiast with local knowledge, contributions are welcome. Check out the repository to see how you can help improve the Texas music community.
