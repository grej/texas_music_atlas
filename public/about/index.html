About Texas Music Atlas

Texas Music Atlas is a volunteer-led, open source project created by a music lover who wants to make it easier to discover songwriter festivals across the state. All data lives in a single JSON file so contributors can keep the calendar fresh without touching a server.

Want to help improve the listings, visuals, or tooling? Fork the repo, open a pull request, or drop a note in the project issues. Every bit of shared knowledge keeps the scene thriving.
